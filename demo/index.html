Spec sheet
  • STT — Deepgram Nova-3 (multi-language Hinglish codemix, interim results, server-side VAD)
  • TTS — Sarvam Bulbul v3, speaker simran (Indian-accent Hinglish, low-latency streaming)
  • LLM — Key-rotating OpenAI-compatible proxy: Groq llama-4-scout (primary, TTFB ~100 ms) → Groq llama-3.3-70b (nuance fallback + hard-no escalation) → Cerebras gpt-oss-120b → Gemini Flash (opt-in)
  • Orchestrator — Pipecat 1.x pipeline with Silero VAD, structured tool calling, and min-words user-turn-start strategy
  • Telephony — Vobiz Media Streams (Indian DID, mu-law 8 kHz, per-call answer_url, 1 h Cloud Run timeout for long sessions)
  • Database — Firestore (asia-south1) — parents, children, courses, demos, call_attempts + turns subcollection, callbacks, objections, rate limits
  • Deployment — Cloud Run (autoscaling, 1 Gi / 1 vCPU, concurrency 8) + Firebase Hosting (Lighthouse 99/100/100/100/100) + Google Secret Manager

Step 1 of 3

What is Aria?

A voice AI counsellor that calls parents within hours of their child's free demo class. Built for Vedantu-style EdTech. The agent speaks Hinglish, handles objections live, and lands a concrete next step.

Step 2 of 3

You roleplay the parent

Pick one of five seeded personas (parent + child + demo subject). Enter your phone. Aria thinks it's calling that parent — pick up and play along.

Step 3 of 3

Watch live monitoring

The Live Monitoring panel shows confidence-scored sentiment, parent tone, and (for soft/hard no) the cited objection plus Priya's response. Try a soft "PW sasta hai" or a hard "interest nahi" — see the difference.
